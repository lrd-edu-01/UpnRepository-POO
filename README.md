# Proyecto POO - Control de Versiones con Git y GitHub

## Descripción
Repositorio creado para desarrollar los casos prácticos del curso de Programación Orientada a Objetos (POO) y aplicar conceptos de control de versiones utilizando Git y GitHub.

## Objetivos
- Gestionar versiones del código mediante Git.
- Utilizar GitHub como repositorio remoto.
- Crear y administrar ramas.
- Realizar merges y Pull Requests.
- Documentar el desarrollo del proyecto.

## Tecnologías utilizadas
- Java
- Eclipse IDE
- Git
- GitHub

## Estructura del proyecto

```
src/
├── sem1_caso1
├── sem1_caso2
├── sem1_caso3
├── sem1_caso4
├── sem1_caso5
```

## Funcionalidades implementadas
- Desarrollo de casos prácticos de Programación Orientada a Objetos.
- Gestión de versiones mediante commits.
- Trabajo con ramas `main` y `develop`.
- Integración de cambios mediante merge.
- Uso de Pull Requests para la integración de funcionalidades.
